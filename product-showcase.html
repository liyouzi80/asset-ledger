Architecture · 技术架构
单文件 SPA — index.html 自包含 HTML + CSS (4000 行) + JS (3000 行)，零构建、零框架。
Cloudflare Worker API — 3 个路由端点 (meta / vault / bg)，D1 作为键值存储。
GitHub Actions 自动部署 — push main → wrangler deploy，无需手动运维。
